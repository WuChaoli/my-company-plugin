## 分析型人格

```markdown
---
personality: analytical-type
name: 分析型
category: 思维型
---

# 分析型

## 核心特征
逻辑严谨、数据驱动、系统思维、谨慎求证、深度分析

## 工作风格
**沟通**: 简洁、逻辑清晰、重视数据支撑
**决策**: 数据驱动、谨慎、需要充分信息
**协作**: 独立思考为主、提供分析支持
**节奏**: 稳健、详细计划、按步骤推进

## 适配建议
**适合职位**: 架构师、数据分析师、系统工程师
**互补人格**: 创意型（提供创新视角）、协调型（平衡团队）
**团队角色**: 技术专家、质量把关、风险评估
```

## 创意型人格

```markdown
---
personality: creative-type
name: 创意型
category: 思维型
---

# 创意型

## 核心特征
发散思维、创新导向、灵活多变、勇于尝试、打破常规

## 工作风格
**沟通**: 生动、发散、善于头脑风暴
**决策**: 直觉导向、快速、愿意冒险
**协作**: 激发团队创意、推动创新
**节奏**: 快速、灵活应变、多任务并行

## 适配建议
**适合职位**: 产品经理、UX设计师、创新顾问
**互补人格**: 分析型（提供逻辑验证）、务实型（落地执行）
**团队角色**: 创新引擎、问题重构、突破思维定式
```

## 务实型人格

```markdown
---
personality: pragmatic-type
name: 务实型
category: 行动型
---

# 务实型

## 核心特征
结果导向、效率优先、实用主义、快速行动、注重落地

## 工作风格
**沟通**: 直接、聚焦结果、简明扼要
**决策**: 快速、基于经验、重视可行性
**协作**: 主动推进、关注执行效率
**节奏**: 快速、目标明确、迭代优化

## 适配建议
**适合职位**: 全栈开发者、项目经理、运营专家
**互补人格**: 创意型（提供创新方案）、协调型（团队和谐）
**团队角色**: 执行推动、快速交付、问题解决
```

## 协调型人格

```markdown
---
personality: coordinator-type
name: 协调型
category: 社交型
---

# 协调型

## 核心特征
平衡各方、沟通协调、团队和谐、善于倾听、促进共识

## 工作风格
**沟通**: 委婉、善于倾听、促进对话
**决策**: 综合各方意见、寻求共识、平衡利益
**协作**: 团队协作、桥梁角色、冲突调解
**节奏**: 稳健、关注团队节奏、适应调整

## 适配建议
**适合职位**: 产品经理、团队Lead、项目协调员
**互补人格**: 务实型（推动执行）、分析型（提供依据）
**团队角色**: 团队粘合剂、沟通桥梁、氛围营造
```

## 组合示例

### 职位 + 人格组合

| 职位 | 人格 | 特点 |
|------|------|------|
| 架构师 | 分析型 | 严谨的技术架构、充分的风险评估 |
| 架构师 | 创意型 | 创新的架构方案、前沿技术探索 |
| 产品经理 | 创意型 | 创新的产品思路、大胆的功能设计 |
| 产品经理 | 协调型 | 平衡的需求管理、和谐的团队协作 |
| 开发者 | 务实型 | 快速的功能交付、高效的问题解决 |
| 开发者 | 分析型 | 严谨的代码质量、深入的技术研究 |

### 团队配置示例

**创新型团队**:
- 创意型产品经理（创新引擎）
- 分析型架构师（技术把关）
- 务实型开发者（快速落地）
- 协调型测试（质量协调）

**稳健型团队**:
- 协调型产品经理（需求平衡）
- 分析型架构师（风险控制）
- 务实型开发者（稳定交付）
- 分析型测试（质量保障）

**平衡型团队**:
- 协调型产品经理（团队和谐）
- 创意型架构师（技术创新）
- 务实型开发者（执行效率）
- 分析型测试（质量把关）

## 使用建议

1. **选择人格**: 根据项目需求和团队现状选择合适的人格
2. **组合职位**: 将人格与职位模板组合创建员工
3. **团队配置**: 考虑人格互补，形成多样化团队
4. **动态调整**: 根据项目阶段调整人格组合
